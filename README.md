# oryx

**oryx** is for people who treat AI as a workshop, not a single chat box. You start from a real question or project, then explore it with several models side by side, in threads you can branch and revisit—instead of losing everything in one long transcript.

Each **project** is a dedicated space: your prompts, files, and conversations stay together so context doesn’t drift. You can run the same idea past different models, compare how they answer, or use structured flows when you want more than a straight Q&A—like weighing options or watching models build on each other’s replies.

**oryx** is also built for **working with other people**. Invite teammates into a project with clear roles, so research, writing, and decision-making can happen in one shared place—not scattered across DMs and separate tools.

If your work benefits from second opinions, debate, and iteration—rather than a single autocomplete answer—**oryx** is meant to feel like the natural next step.
